Затем нужно выяснить, где находится на вашем компьютере директория с данными программы MyTetra. Запускаем MyTetra, нажимаем "Инструменты" -> "Настройки" -> "Основные". В поле "Директория с данными" нажимаем кнопочку с тремя точками [...] и смотрим, где расположена директория /data. Все дальнейшие действия из консоли следует выполнять внутри директории /data


Пример тут находится папка программы mytetra data в linux :


/root/.config/mytetra/data


Итак, на втором компьютере, где мы хотим сделать синхронизацию,

инсталлируем MyTetra, выясняем месторасположение каталога /data (как

это сделать, написано выше). Выходим из MyTetra, затем заходим в каталог

/data, и всё в нём удаляем.


rm -r base/ database.ini icons/ mytetra.xml # удаляем все в папке data


Затем, находясь в каталоге /data, даем команду:


ТОЧКА ОБЯЗАТЕЛЬНА В КОНЦЕ!!!
